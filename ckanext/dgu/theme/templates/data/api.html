CKAN API Notes

Developers can access the data catalogue with an API, as an alternative to the data.gov.ie web interface. The API provides both RESTful and functional interfaces, all in JSON format, making it suitable for a wide range of clients.

The base URL for the API is: https://data.gov.ie/api

The API is provided by the CKAN software and full instructions are provided here: http://docs.ckan.org/en/ckan-2.2.3/api.html.
